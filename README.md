# Hydrophobicity Space

A visualization of amino acid hydrophobicity across 23 published hydrophobicity scales, scraped from [ExPASy ProtScale](https://web.expasy.org/protscale/).

## Overview

Different hydrophobicity scales assign different values to amino acids depending on the experimental method used (partitioning, chromatography, structural analysis, etc.). This project aggregates 23 scales and normalizes them to a common 0–100 range (where 100 = most hydrophobic).
